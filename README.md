# stone-paper-scissor-game
Rock Paper Scissors Game 🎮
A simple Rock Paper Scissors game built using JavaScript, HTML, and CSS. The player competes against the computer, which makes a random choice.

✨ Features

Three choices: Rock, Paper, or Scissors

Computer selects randomly

Background color changes based on the result (Win/Loss/Draw)

Minimal and easy-to-play interface

🚀 How to Play

Click on Rock, Paper, or Scissors.

The computer makes a random selection.

The result is displayed, and the background color changes
